# latex-thesis

### Prerequisite

1. Install Docker [https://www.docker.com/toolbox](https://www.docker.com/toolbox)

### Install

1. Launch Docker terminal

2. Go to the folder of this project

3. Run `docker build -t texlive:full .`

### Compile

1. Launch Docker terminal

2. Go to the folder of this project

3. Run `./latex.sh` (every time you want)
